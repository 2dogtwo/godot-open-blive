All notable changes to this project will be documented in this file.

## [2.2.0] - 2022-07-03
### Fixed
- 修正互动玩法心跳间隔
- 调整 API Nonce 的生成算法，尽可能避免重复请求错误

## [2.1.1] - 2022-07-01
### Added

### Fixed
- 修复弹幕服务器无法重连的问题

## [2.1.0] - 2022-06-07
### Added
- 添加获取主播信息接口

## [2.0.0] - 2022-06-06
### Added
- 适配 v2 版本 API，使用主播身份码

## [1.0.0] - 2022-05-30
### Added
